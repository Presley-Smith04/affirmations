# Affirmation Generator

**Affirmation Generator** is a web app that dynamically generates positive and negative affirmations based on user mood and category selection. Designed with a responsive UI and interactive features for engagement.  

## Highlights / Achievements

- Implemented a mood based suggestion system using **JavaScript** for dynamic category selection.  
- Developed dual modes (**Angel & Devil**) to generate positive or negative affirmations using **Flask** and JSON data.  
- Made an interactive UI with **dynamic DOM manipulation**, animated particle effects, and toggleable background music (also made by me).  
- Prioritized smooth user experience with fade in animations and theme switching between modes.  
- Deployed on **Render** using **Gunicorn** for hosting.  

## Tools & Technologies

- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** Python, Flask  
- **Data:** JSON files for affirmation content  
- **Deployment:** Render, Gunicorn  

## Usage

1. Enter your name and mood.  
2. Select or use suggested category.  
3. Toggle between Angel and Devil modes for positive or humbling affirmations.  
4. Click “Generate Another” to refresh affirmation content.
